# Python Assignment

This repository contains Python assignments demonstrating the use of **Python classes**.

## 📁 Files in this repository

1. **mi_players.py**  
   - Demonstrates a `Player` class  
   - Contains 11 MI cricket team players  
   - Each player has attributes: `jersey_no`, `name`, `runs`, `wickets`, `team_name`  
   - Method `display()` prints player details  

2. **unique_classes.py**  
   - Contains 5 unique classes: **Fan, Table, TV, Laptop, Pen**  
   - Each class has at least 2 attributes and 1 method  
   - Demonstrates creating objects and calling methods
